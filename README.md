# OkenSDK_iOS
iOS SDK 


***To get access key, please visit https://oken.tech and contact us!***

To install OkenSDK, please, use cocoapods

```
use_frameworks!

platform :ios, '13.0'

target 'testsdk' do

 pod 'OkenSDK', :git => 'https://github.com/OkenDev/OkenSDK_iOS.git'
end
```
